### API's

Se cosumieron dos API´s públicas para estas dos app´s. Para la primera aplicación (rama master) se consumió de la API que provee [GeoNames](http://www.geonames.org/), se utilizó para obtener una lista de ciudades que el usuario busca.

Para la segunda app (rama desarrollo2) se utilizó la API que provee [OpenWeatherMap](https://openweathermap.org/), ésta obtiene la información sobre del clima de la ubicación que se le manda como parámetro.
